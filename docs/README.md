# docs/

Design documentation for coding swarm concepts. Subdirectories mirror the hub files in the project root.

## Subdirectories

- `architecture/` — conceptual architecture, C4 models, DDD context maps
- `ubiquitous-language/` — domain vocabulary by bounded context
- `tech-stack/` — tooling and diagram conventions
- `developer-workflows/` — design authoring and review workflows
- `user-experience/` — user journeys, interaction patterns, UX principles
- `adr/` — numbered architecture decision records
- `musings/` — exploratory notes and informal thinking
